Dados e backup
O que é: Tabela com todos os abastecimentos (data, equipamento, litros, horímetro).
Para que serve: Abrir no Excel ou Google Planilhas para análise, relatórios e conferência.
Como usar: Toque em "Exportar" → arquivo .csv é baixado → abra no Excel (Arquivo → Abrir) ou arraste para o Google Drive.
O que é: Cópia completa de todos os dados — equipamentos, operadores, entradas, comboios e abastecimentos.
Para que serve: Guardar backup de segurança ou transferir para outro dispositivo.
Como usar: Toque em "Backup" → arquivo .json é baixado → guarde em local seguro (iCloud, Google Drive, e-mail). Para restaurar, use "Importar" abaixo.
O que é: Restaura um backup salvo anteriormente.
Atenção: Substitui todos os dados atuais pelo conteúdo do arquivo.
Como usar no iPhone: ① No Mac, toque em "Backup completo" → salve o arquivo. ② Envie por AirDrop, iCloud Drive, WhatsApp ou e-mail para o iPhone. ③ No iPhone, toque em "Importar" → selecione o arquivo recebido → confirme.
